# Risk Assessment Template (Markdown)

This repository contains a simple, reusable **Risk Assessment Template** in Markdown format. It is designed for GRC, audit, or compliance teams to clone, customize, and use directly within GitHub or any text editor.

## How to Use

1. Click **Use this template** or **Download ZIP**
2. Copy `template.md` and rename it for each risk (e.g., `Risk-Slack-API.md`)
3. Fill out each section using plain text or VS Code preview mode
4. Store versions and comments via Git
